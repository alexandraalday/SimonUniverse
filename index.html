Instructions

  1. Press the START button. You will receive the first signal. Repeat the signal by pressing the same color gem.
  2. The computer will duplicate the symbol and add one. Repeat these two signals by pressing the same color gems, in order.
  3. The computer will duplicate these first two signals and add one.
  4. Continue playing as long as you can repeat each sequence of signals correctly.
  5. If you select Normal Mode, when you fail to repeat a sequence exactly, the computer will respond with a "RAZZ" sound. This means you are incorrect and will receive teh same sequence and a second try to repeat the correct sentence.
  6. If you select Crystal Gem mode, when you fail to repeat a sequence exactly, the computer will respond with a "RAZZ" sound. This means you lost, and the game will start over at round 1.
