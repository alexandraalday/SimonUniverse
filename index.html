Instructions

  • ☆ Press the START button. You will receive the first signal. Repeat the signal by pressing the same color gem.
  • ☆ The computer will duplicate the symbol and add one. Repeat these two signals by pressing the same color gems, in order.
  • ☆ The computer will duplicate these first two signals and add one.
  • ☆ Continue playing as long as you can repeat each sequence of signals correctly.
  • ☆ If you select Normal Mode, when you fail to repeat a sequence exactly, the computer will respond with a "RAZZ" sound. This means you are incorrect and will receive teh same sequence and a second try to repeat the correct sentence.
  • ☆ If you select Crystal Gem mode, when you fail to repeat a sequence exactly, the computer will respond with a "RAZZ" sound. This means you lost, and the game will start over at round 1.
